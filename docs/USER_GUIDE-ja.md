## 主要機能

### チャットインターフェース

Moca のメイン画面は、シンプルで直感的なチャットインターフェースです。

![メイン画面](./assets/agentchat.geeawa.net_chat.png)

#### 主な機能

- **質問・相談**: AIに質問したり、相談したりできます
- **文章の添削**: 文章の校正や改善案を受け取れます
- **Web検索**: Web検索から最新の情報を含んだ回答を得られます
- **要約の作成**: 長い文章の要約を生成できます
- **ブレインストーミング**: アイデア出しのサポートを受けられます
- **比較・分析**: 複数の選択肢を比較・分析できます

#### メッセージの送信

1. 画面下部のテキストボックスに質問やメッセージを入力します
2. **Enter**キーを押すと送信、**Shift + Enter**で改行できます
3. 右下の送信ボタン（紙飛行機アイコン）をクリックしても送信できます

#### モデルの選択

画面下部のドロップダウンメニューからAIモデルを選択できます。
- デフォルト：**Claude Sonnet 4.5 (Anthropic)**

### AIエージェント管理

Moca では、様々な分野に特化したAIエージェントを選択・作成・カスタマイズできます。

#### エージェントの選択

![エージェント選択](./assets/agentchat.geeawa.net_chat_select_agent.png)

左サイドバーのアイコンをクリックして「エージェント選択」を開き、利用可能なエージェントのリストを表示します。

##### 利用可能なエージェント

| エージェント名 | 説明 |
|-----------|-------------|
| **General Assistant** | 一般的な質問やタスクをサポートする基本エージェント |
| **Software Developer** | GitHub統合を持つソフトウェア開発エージェント |
| **Code Review Agent** | コードレビューとプログラミング支援に特化したエージェント |
| **Data Analyst Agent** | データ分析と統計処理に特化したエージェント |
| **PowerPoint Creator** | プレゼンテーション資料作成に特化したエージェント |
| **Web Deep Researcher** | 深い調査と情報収集に特化したエージェント |
| **Physicist** | 理論物理学と計算物理学に特化したエージェント |
| **Image Creator** | Amazon Nova Canvasを使用した画像生成エージェント |
| **Slideshow Video Creator** | 日本語字幕とナレーション付き動画作成エージェント |
| **Kamishibai Master** | 日本の紙芝居作成に特化したエージェント |
| **Knowledge Base Search Agent** | Amazon Bedrockを使用したセマンティック検索エージェント |

#### 新しいエージェントの作成

1. エージェント選択画面の「**+ Create New Agent**」ボタンをクリック
2. エージェント名、説明、アイコンを設定
3. 必要なツールを選択
4. 保存して完了

### ツール管理

様々なツールを追加して、エージェントの機能を拡張します。

![ツール選択](./assets/agentchat.geeawa.net_chat_select_tool.png)

#### ツールの追加・編集

1. エージェント選択画面でエージェントメニュー（...）を開く
2. 「編集」を選択
3. 「ツール」タブを選択
4. 利用可能なツールリストから必要なツールを選択
5. 「保存」をクリック

#### 主な利用可能ツール

| ツール名 | 説明 |
|----------|-------------|
| **execute_command** | シェルコマンドを実行。ファイル操作、情報収集、開発タスク自動化に使用 |
| **tavily_search** | Tavily APIを使用した高品質なWeb検索。最新情報やニュースを取得 |
| **tavily_extract** | 指定されたURLからコンテンツを抽出。Webページから構造化されたテキストを取得 |
| **file_editor** | ファイルの読み書きと編集操作 |
| **nova_canvas** | Amazon Nova Canvasを使用した画像生成 |
| **s3_list_files** | S3ストレージからファイルリストを取得 |

### ファイルストレージ

ファイルをアップロードして管理するクラウドストレージ機能です。

![ファイルストレージ](./assets/agentchat.geeawa.net_chat_file_storage.png)

#### 主な機能

- **ファイルアップロード**: 最大5MBまでのファイルをアップロード
- **フォルダ作成**: フォルダを作成してファイルを整理
- **ファイル管理**: アップロードしたファイルのダウンロードと削除

#### 使い方

1. 左サイドバーの「ファイル管理」アイコンをクリック
2. 「ファイルストレージ」パネルが開きます
3. **アップロード**ボタンをクリックしてファイルをアップロード
4. **新しいフォルダ**ボタンをクリックして新しいフォルダを作成
5. ファイル名の横のアイコンを使用してファイルをダウンロードまたは削除

**注意**: ファイルサイズの上限は5MBです。

### エージェント共有

組織内で作成されたエージェントを検索して使用できます。

![エージェント検索](./assets/agentchat.geeawa.net_chat_share_agent.png)

#### 使い方

1. 左サイドバーの「検索」アイコンをクリック
2. 「エージェント検索」ページが開きます
3. 検索ボックスにキーワードを入力して検索
4. エージェントをクリックして詳細を表示
5. 「マイエージェントに追加」をクリックしてエージェントリストに追加

## 専門エージェントの例

### Physicist（物理学エージェント）

理論物理学と計算物理学に特化したエージェントです。複雑な物理現象のシミュレーションや比較分析が可能です。

![Physicist](./assets/agentchat.geeawa.net_chat_physicist.png)

#### 機能

- 物理現象のシミュレーション
- 減衰振動の比較分析
- グラフによる可視化
- 理論値と測定値の比較

上のスクリーンショットは、3つの減衰パターン（不足減衰、臨界減衰、過減衰）の比較分析を示しています。

---

### Slideshow Video Creator（スライドショー動画作成）

PDFファイルから日本語字幕とナレーション付き動画を自動生成するエージェントです。

![PDF to Video](./assets/agentchat.geeawa.net_chat_pdf-to-video.png)

#### 処理フロー

1. PDFファイルをアップロード
2. AIが各PDFページを画像に変換
3. 日本語字幕とナレーションを生成
4. 動画として出力

![Video Result](./assets/agentchat.geeawa.net_chat_pdf-to-video-result.png)

#### 使用例

```
PDFファイル（PowerPointエクスポート）をナレーション付き動画に変換してください：
PDFファイル: /20251019 Context engineering.pdf
```

エージェントは以下を自動的に実行します：
- PDFページを画像に変換
- 字幕を生成
- ナレーション音声を生成
- 動画を結合して出力

## セッション管理

チャットセッション（会話履歴）を管理します。

### 主な機能

- **セッションリスト**: 過去の会話履歴を表示
- **セッション切り替え**: 以前の会話に戻って続きを行う
- **セッション削除**: 不要な会話履歴を削除
- **新しいセッション**: 新しい会話を開始

### 使い方

1. 左サイドバー上部の「ホーム」アイコンをクリック
2. セッションリストが表示されます
3. セッション名をクリックして切り替え
4. 「...」メニューで削除などの操作を実行
